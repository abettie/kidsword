# プロジェクト概要

このプロジェクトは、インフラ、フロントエンド、バックエンドを含むモノレポ構成です。

## プロジェクト構成

```
/
├── infrastructure/    # インフラ
├── frontend/          # フロントエンド
└── backend/           # バックエンド
```

## 技術スタック

- **インフラ**: Terraform (AWS Lambda, DynamoDB, etc.)
- **フロントエンド**: React, TypeScript
- **バックエンド**: TypeScript, AWS Lambda, DynamoDB
